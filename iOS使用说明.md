# iOS 使用说明

## 方法一：使用 Safari 浏览器（推荐）

1. **将HTML文件保存到 iCloud Drive 或文件应用**
   - 通过 AirDrop、邮件或云盘将 `窗帘报价计算器_手机版.html` 传到 iPhone
   - 在"文件"应用中打开

2. **用 Safari 打开**
   - 在文件应用中点击 HTML 文件
   - 选择"在 Safari 中打开"
   - 或者长按文件，选择"共享" → "在 Safari 中打开"

3. **添加到主屏幕（推荐）**
   - 在 Safari 中打开后，点击底部的"分享"按钮
   - 选择"添加到主屏幕"
   - 这样以后就可以像 App 一样直接打开使用

## 方法二：使用在线版本（最简单）

### 方案A：使用 GitHub Pages（免费）

1. 将文件上传到 GitHub 仓库
2. 启用 GitHub Pages
3. 通过网址访问，可以添加到主屏幕

### 方案B：使用其他免费托管服务

- **Netlify Drop**: 直接拖拽HTML文件即可获得网址
- **Vercel**: 上传文件获得网址
- **CodePen**: 在线编辑和分享

## 方法三：使用本地服务器（开发者）

如果你有 Mac 电脑：

1. 在 Mac 上打开终端
2. 进入文件所在目录
3. 运行：`python3 -m http.server 8000`
4. 在 iPhone Safari 中访问：`http://你的Mac的IP地址:8000/窗帘报价计算器_手机版.html`

## 方法四：使用文件应用 + Safari

1. 将 HTML 文件保存到"文件"应用
2. 在"文件"应用中找到文件
3. 长按文件 → 选择"共享" → "在 Safari 中打开"
4. 添加到主屏幕以便快速访问

## 为什么 Chrome 在 iOS 上无法打开本地文件？

iOS 上的 Chrome 实际上使用的是 Safari 的 WebKit 引擎，并且受到 iOS 系统限制，无法直接访问本地文件系统。这是 iOS 的安全策略。

## 最佳实践

**推荐流程：**
1. 将 HTML 文件保存到 iCloud Drive
2. 在 iPhone 的"文件"应用中打开
3. 用 Safari 打开
4. 添加到主屏幕，以后像 App 一样使用

## 注意事项

- 导出数据功能需要浏览器支持下载，Safari 完全支持
- 保存为图片功能需要 html2canvas 库，已通过 CDN 加载，需要网络连接
- 建议添加到主屏幕后，即使离线也可以使用基本功能
